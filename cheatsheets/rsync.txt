# cpr
Copy with progress bar (rsync, resumable)
cpr <source> <destination>
---
# mvr
Move with progress bar (rsync, then delete source)
mvr <source> <destination>
---
# backup-dev
Backup ~/Developers to external drive (excludes node_modules, vendor, .git)
backup-dev <destination>
---
# backup-sync
Backup any directory with smart excludes
backup-sync <source> <destination>
---
# deploy
Deploy project to remote server via SSH
deploy <local-path> <user@host:remote-path>
---
# deploy-laravel
Deploy Laravel project (excludes vendor, storage/logs)
deploy-laravel <user@host:remote-path>
---
# deploy-node
Deploy Node.js project (excludes node_modules, .next, dist)
deploy-node <user@host:remote-path>
---
# sync-push
Push local directory to another machine
sync-push <host> [directory]
---
# sync-pull
Pull directory from another machine
sync-pull <host> [directory]
---

rsync-help
