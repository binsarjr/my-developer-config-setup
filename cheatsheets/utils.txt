# mkcd
Create directory and cd into it
mkcd <directory>
---
# backup
Create timestamped backup of file (file.bak.YYYYMMDD_HHMMSS)
backup <file>
---
# extract
Universal archive extractor (tar.gz, zip, rar, 7z, etc.)
extract <archive>
---
# dlx
Download and extract file in one command
dlx <url>
---
# ports
Show all listening ports
ports
---
# myip
Show public IP address
myip
---
# localip
Show local IP address (en0)
localip
---
# weather
Show weather for current location or city
weather [city]
---
# project-analyze
Analyze project and recommend tools (detects Node.js, Laravel)
project-analyze [dir]
---
